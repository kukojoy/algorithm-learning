## **datetime模块**

```python
import datetime
from datetime import datatime, timedelta
a = datetime.date(2024, 3, 17) # 输入日期
a = datetime(2024, 3, 17) # 输入日期
b = datetime.timedelta(days = 1) # 输入时间间隔
c = a + b
```
